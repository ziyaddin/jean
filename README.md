## Setup

<p>Open your <code>Terminal</code> and type:</p>

```bash
git clone git@github.com:ziyaddin/jean.git
cd jean
sh installer
```

## Install Packages

<p></p>

```bash
jean install <package-name>
```

## Initialize Scripts

<p></p>

```bash
jean init <script-name>
```

## Search Packages

<p></p>

```bash
jean search
```

It will search packages in <code>packages.txt</code>

## Help

<p></p>

```bash
jean help
```
